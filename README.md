# adzuki-beans-soup
紅豆湯食譜

# Pre-requisite 道具準備
1. 篩網
2. 電鍋
3. 錫箔紙
4. 筷子

# Preparation 事前準備
> 紅豆:水 是 1:3

1. 紅豆600g
2. 水1800g (紅豆湯)
3. 水360ml (電鍋用) x 2
4. 鹽巴1小匙
5. 二級砂糖200g

# Procedure 步驟
1. 將紅豆倒入**篩網**，並用清水沖洗。
2. 將清洗後的紅豆倒入**電鍋內鍋**，並加入水。
3. 將**電鍋內鍋**放到瓦斯爐上煮，水滾後浮上來的紅豆是不好的豆子，用湯匙挑掉。
4. 水滾後約2分鐘後關火，並用**錫箔紙**將頂部封住。
5. 將水倒入電鍋內，並將**電鍋內鍋**放進去，按下開關。
6. 按鈕跳起後，再悶20分鐘。
7. 撕開錫箔紙一個小洞，並用**筷子**輕輕攪動紅豆。
8. 再將水倒入電鍋。按下開關。
9. 按鈕跳起後，再悶20分鐘。
10. 加入鹽，加入糖，攪拌
